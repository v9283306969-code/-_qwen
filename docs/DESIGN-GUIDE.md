# Design Guide — MLECP

Основано на сайте Add Me (add-me.shop) — ювелирные изделия, золото.

## Типографика

| Элемент | Шрифт | Вес | Где используется |
|---|---|---|---|
| Заголовки (h1-h3) | Playfair Display | 400, 600, 700 | Hero, секции, карточки товаров |
| Текст | Inter | 400, 500, 600 | Описания, кнопки, навигация |
| Код | JetBrains Mono | 400 | Технические блоки (future) |

Оба шрифта поддерживают кириллицу. Google Fonts.

## Принципы дизайна

1. **Слоновая кость — основа** — тёплый, мягкий фон `rgb(253,252,243)`, создаёт ощущение натуральности
2. **Золото `#B8933B` — приглушённое** — не кричит, не блестит — «благородная патина»
3. **Описания — чёрный `#000000`** — максимальная читаемость, контраст на слоновой кости
4. **Без теней** — плоские карточки, разделение через линии и отступы, не через тени
5. **Строгие линии, без закруглений** — `border-radius: 0` везде. Прямоугольные карточки, кнопки, инпуты
6. **Минимум декора** — воздух между элементами важнее, чем декор
7. **Чёрный + золотой заголовок** — чередуются для иерархии (чёрный — основной раздел, золотой — акцентный)

## Рекомендации для премиального ощущения

Эти пункты не на референсе, но рекомендуются для ювелирного e-commerce с MLM-сетью:

| Элемент | Предложение | Почему |
|---|---|---|
| **Hero-секция** | Полноширинное фото изделия + чёрный заголовок по центру | Как editorial в журнале — первое впечатление «это дорого» |
| **Карточки товаров** | Фото на слоновой кости (не белый фон карточки) | Единый фон — изделие «парит», нет рамки/бордюра |
| **Кнопка «Купить»** | Золотой фон `#B8933B`, чёрный текст, без закруглений | Золотая кнопка на слоновой кости — единственный цветной элемент страницы |
| **Hover на карточках** | Лёгкая золотая линия-подчёркивание снизу (1px `#B8933B`) | Не тень, не подсветка — строгая линия, как в печатном каталоге |
| **Навигация** | Тонкая золотая линия-разделитель после хедера | Визуальная опора, не перегружает |
| **Фото товаров** | На слоновой кости или на чёрном фоне (чередовать для разных категорий) | Кольца — на слоновой кости, часы — на чёрном |
| **Разделители секций** | Тонкая горизонтальная линия `#D4B56A` (30% opacity), 60px отступы | Пространство важнее линий |
| **Шрифт заголовков** | Playfair Display, weight 400 (не 700!) | Тонкий serif = премиум. Жирный serif = логотип кафе |
| **Логотип** | Текстовый, Playfair Display, золотой | Без иконки — текст дороже картинки |
| **Партнёрский дашборд** | Минимум графиков — «ваша сеть: X человек», «доход: Y», таблица без декора | Партнёры видят цифры, не UI-украшения |

## shadcn/ui Theme Config

Для `apps/web/app/globals.css`:

```css
@import "tailwindcss";

@theme inline {
  --font-sans: "Inter", sans-serif;
  --font-serif: "Playfair Display", serif;
  --font-mono: "JetBrains Mono", monospace;

  /* Слоновая кость основной фон */
  --color-background: #FDFCF3;
  --color-foreground: #000000;

  /* Карточки */
  --color-card: #FFFFFF;
  --color-card-foreground: #000000;

  /* Золото */
  --color-primary: #B8933B;
  --color-primary-foreground: #000000;

  /* Вторичный — слоновая кость темнее */
  --color-secondary: #F5EFD5;
  --color-secondary-foreground: #000000;

  /* Muted */
  --color-muted: #E8E0C8;
  --color-muted-foreground: #444444;

  /* Акцент */
  --color-accent: #B8933B;
  --color-accent-foreground: #000000;

  /* Границы — тонкие, без закруглений */
  --color-border: #D4B56A40;
  --color-input: #D4B56A40;
  --color-ring: #B8933B;
  --radius: 0;

  /* Статусы (приглушённые) */
  --color-success: #4A9E6D;
  --color-warning: #C4943A;
  --color-error: #C44B3C;
  --color-info: #5B7FA5;
}

/* Глобальный сброс закруглений */
* {
  border-radius: 0 !important;
}

/* Убираем тени */
.shadow, .shadow-sm, .shadow-md, .shadow-lg, .shadow-xl {
  box-shadow: none !important;
}
```

**Dark theme** (опционально, через `app/globals.css` при `prefers-color-scheme: dark`):

```css
@media (prefers-color-scheme: dark) {
  @theme inline {
    --color-background: #1C1C1C;
    --color-foreground: #F0EDE3;
    --color-card: #2A2A2A;
    --color-card-foreground: #F0EDE3;
    --color-primary: #B8933B;
    --color-primary-foreground: #000000;
  }
}
