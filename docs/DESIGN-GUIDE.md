### Текст

| Роль | Light Mode | Dark Mode | Где используется |
|---|---|---|---|
| **Чёрный заголовок строки** | `#000000` | `#FDFCF3` | «ДОБАВЬ СЕБЕ ЭНЕРГИИ И ЗДОРОВЬЯ» — основной контент |
| **Золотой секционный заголовок** | `#B8933B` | `#D4B56A` | «СПОРТИВНОЕ ПИТАНИЕ», «СЕРТИФИКАТЫ» — навигационные разделители |
| **Золотой подзаголовок категории** | `#B8933B` | `#D4B56A` | «БАДЫ», «ПРЕИМУЩЕСТВА» — визуальная пауза |

| Muted (даты, подпись) | `#777777` | `#666666` | — |

Оба шрифта поддерживают кириллицу. Google Fonts.

## Принципы дизайна

1. **Слоновая кость — основа** — тёплый, мягкий фон `rgb(253,252,243)`
2. **Золото `#B8933B` — приглушённое** — не кричит — «благородная патина»
3. **Чередование заголовков** — чёрный (контент) → золотой (разделитель) → чёрный → золотой. Ритм как в журнале
4. **Золотые иконки (ромбики ✦)** — перед списками преимуществ, вместо буллитов
5. **Золотые рамки карточек** — тонкая линия `1px solid #B8933B`, без теней
6. **Без теней** — плоские элементы, разделение через линии и отступы
7. **Строгие линии, без закруглений** — `border-radius: 0`
8. **Минимум декора** — воздух важнее оформления

## Рекомендации для премиального ощущения

На основе скриншотов с add-me.shop + рекомендации для ювелирного e-commerce:

| Элемент | Как на сайте | Рекомендация для MLECP |
|---|---|---|
| **Hero-секция** | Крупный чёрный заголовок | Для ювелирки — полноширинное фото + чёрный заголовок |
| **Секционные заголовки** | Крупный золотой (`СПОРТИВНОЕ ПИТАНИЕ`) | Для ювелирки — золотой заголовок секции (`КАТАЛОГ`, `КОЛЛЕКЦИИ`) |
| **Карточки преимуществ** | Золотая рамка + ромбик ✦ | Для ювелирки — те же карточки для УТП (сертификаты, доставка) |
| **Карточки товаров** | Фото + чёрный заголовок + описание | Для ювелирки — крупное фото, минимум текста |
| **Кнопки** | — | Золотой фон `#B8933B`, чёрный текст, без закруглений |
| **Навигация** | — | Тонкая золотая линия-разделитель после хедера |
| **Партнёрский дашборд** | — | Чёрный заголовок строки + золотой разделитель секций |

## shadcn/ui Theme Config

Для `apps/web/app/globals.css`:

```css
@import "tailwindcss";

@theme inline {
  --font-sans: "Inter", sans-serif;
  --font-serif: "Playfair Display", serif;
  --font-mono: "JetBrains Mono", monospace;

  /* Слоновая кость основной фон */
  --color-background: #FDFCF3;
  --color-foreground: #000000;

  /* Карточки */
  --color-card: #FFFFFF;
  --color-card-foreground: #000000;

  /* Золото */
  --color-primary: #B8933B;
  --color-primary-foreground: #000000;

  /* Вторичный — слоновая кость темнее */
  --color-secondary: #F5EFD5;
  --color-secondary-foreground: #000000;

  /* Muted */
  --color-muted: #E8E0C8;
  --color-muted-foreground: #444444;

  /* Акцент */
  --color-accent: #B8933B;
  --color-accent-foreground: #000000;

  /* Границы — тонкие, без закруглений */
  --color-border: #D4B56A40;
  --color-input: #D4B56A40;
  --color-ring: #B8933B;
  --radius: 0;

  /* Статусы (приглушённые) */
  --color-success: #4A9E6D;
  --color-warning: #C4943A;
  --color-error: #C44B3C;
  --color-info: #5B7FA5;
}

/* Глобальный сброс закруглений */
* {
  border-radius: 0 !important;
}

/* Убираем тени */
.shadow, .shadow-sm, .shadow-md, .shadow-lg, .shadow-xl {
  box-shadow: none !important;
}
```

    --color-foreground: #F0EDE3;
    --color-card: #2A2A2A;
    --color-card-foreground: #F0EDE3;
    --color-primary: #B8933B;
    --color-primary-foreground: #000000;
  }
}
