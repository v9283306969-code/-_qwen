# Design Guide — MLECP

Основано на сайте Add Me (add-me.shop) — ювелирные изделия, золото.

## Цветовая палитра

| Роль | Имя | Hex | Tailwind CSS |
|---|---|---|---|
| Primary (Золотой) | `gold-500` | `#D4A853` | Основной акцент бренда |

## Типографика

| Элемент | Шрифт | Вес | Где используется |
|---|---|---|---|
| Заголовки (h1-h3) | Playfair Display | 400, 600, 700 | Hero, секции, карточки товаров |
| Текст | Inter | 400, 500, 600 | Описания, кнопки, навигация |
| Код | JetBrains Mono | 400 | Технические блоки (future) |

Оба шрифта поддерживают кириллицу. Google Fonts.

## Принципы дизайна

1. **Премиальность через минимализм** — много пустого пространства, мало декоративных элементов
2. **Золото — акцент, не фон** — золото только для кнопок, иконок, акцентных линий
3. **Тёмная тема — приоритет** — для ювелирки тёмный фон продаёт лучше, золото «светится»
4. **Карточки товаров — крупные фото** — минимальный оверлей, максимум продукта
5. **CTA — контрастное золото на тёмном** — кнопки покупки всегда видны
